# README images

`screenshot.png` is a placeholder banner. Replace it with a real screenshot
of the app to showcase it in the top-level `README.md`.

To capture a clean window shot on macOS:

1. Open Rewind and get it to a good screen (e.g. a game hub with match cards).
2. Press **⌘⇧4**, then **Space**, then click the Rewind window.
3. Save/move the result to `docs/images/screenshot.png` (overwrite this
   repo's placeholder).

(The agent couldn't capture it automatically — `screencapture` needs Screen
Recording permission for the calling terminal, which the sandboxed shell
doesn't have.)
